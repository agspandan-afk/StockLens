How to use — 3 steps

01
Get your stock data
Go to finance.yahoo.com → search ticker (e.g. TATASTEEL.NS) → Historical Data → Download. Or copy from NSE / Moneycontrol.
02
Paste the CSV
Open the downloaded file in Notepad, select all, copy and paste below. The app reads Date + Close columns automatically.
03
Hit Analyse
All charts, signals, forecast and roadmap run instantly in your browser. No internet needed after this page loads.
